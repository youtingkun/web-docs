# ls 命令

语法格式: `ls [参数] [文件]`

| 参数    | 描述                                             |
| ------- | ------------------------------------------------ |
| -a      | 显示所有文件及目录 (包括以“.”开头的隐藏文件)     |
| -l      | 使用长格式列出文件及目录的详细信息               |
| -r      | 将文件以相反次序显示(默认依英文字母次序)         |
| -t      | 根据最后的修改时间排序                           |
| -A      | 同 -a ，但不列出 “.” (当前目录) 及 “..” (父目录) |
| -S      | 根据文件大小排序                                 |
| -R      | 递归列出所有子目录                               |
| -d      | 查看目录的信息，而不是里面子文件的信息           |
| -i      | 输出文件的 inode 节点信息                        |
| -m      | 水平列出文件，以逗号间隔                         |
| -X      | 按文件扩展名排序                                 |
| --color | 输出信息中带有着色效果                           |
